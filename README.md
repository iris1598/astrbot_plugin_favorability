# astrbot_plugin_favorability

好感度系统插件，将 AI 与用户之间的互动量化为好感度分数与评价，支持表情包回应。

## 功能

- **LLM 注入**：每次对话前将当前好感度状态注入 system prompt，让 AI 根据内心状态调整回复风格。
- **好感度解析**：解析 AI 回复中的 `[FAV:±数值]`、`[EVAL:评价]`、`[STK:分类]` 标记，自动更新数据、发送表情包。
- **指令查询**：用户可查询自己的好感度档案与排行榜，管理员可强制设置/查询/重置指定用户。
- **表情包支持**：在数据目录 `data/plugin_data/favorability/stickers/<分类名>/` 下放置图片（运行后自动创建），AI 可发送对应表情包。

## 标记格式

AI 回复末尾可选输出：

```
[FAV:+3]          → 好感度 +3（范围 -5 ~ +5）
[EVAL:很有趣的人]  → 更新对该用户的评价
[STK:开心]        → 发送"开心"分类下的随机表情包
```

## 表情包目录结构

```
data/plugin_data/favorability/stickers/
├── 开心/
│   ├── 1.png
│   └── 2.gif
└── 生气/
    └── 1.jpg
```

## 配置项

| 配置 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `session_based` | bool | false | 开启后各群/私聊好感度独立，关闭则按群号+用户ID划分 |

## 指令

| 指令 | 权限 | 说明 |
|------|------|------|
| `/好感度查询` | 全员 | 查看自己的好感度和评价 |
| `/我的好感度` | 全员 | 同上（别名） |
| `/好感度排行` | 全员 | 查看当前会话排行榜 |
| `/重置好感度` | 全员 | 重置自己的记录 |
| `/设置好感度 <ID> <分>` | 管理员 | 强制设定好感度 |
| `/查询好感度 <ID>` | 管理员 | 查询指定用户 |
| `/重置指定好感度 <ID>` | 管理员 | 重置指定用户 |
