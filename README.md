# astrbot_plugin_favorability

好感度系统插件——AI 根据对话内容自主更新用户好感度与评价，支持表情包回应、禁言处罚、PIL 图片渲染查询与排行。

## 功能

- **好感度系统**：LLM 根据对话内容自动调整用户好感度（+1 ~ -5 不等），并根据好感度改变行为态度
- **印象系统**：LLM 自动生成对用户的简短印象/评价
- **表情包回应**：LLM 可根据情绪选择合适的表情包发送
- **禁言机制（v2.1.0+）**：LLM 可对持续恶劣行为的用户输出 `[MUTE:N]` 标签进行禁言处罚，最长 5 分钟阻断 LLM 回复
- **PIL 图片渲染**：好感度查询和排行榜结果以精美的图片形式展示
- **会话隔离**：每个群/私聊的好感度独立计算

## 架构

```
astrbot_plugin_favorability/
├── main.py            — 插件入口，路由指令
├── models/manager.py  — 数据层：好感度 CRUD、禁言管理与持久化
├── services/
│   ├── sticker.py     — 表情包目录管理

├── llm/handler.py     — LLM 请求注入、禁言拦截与响应解析
├── commands/
│   ├── user.py        — 用户指令（查询/排行/重置）
│   └── admin.py       — 管理员指令（设置/重置/禁言/解除禁言）
├── render/image.py    — PIL 图片渲染（查询卡片/排行榜）
└── metadata.yaml      — 插件元数据
```

## 指令

### 用户指令
| 指令 | 说明 |
|------|------|
| `/查询好感度 [@用户]` | 查看自己或@他人的好感度和评价（PIL 图片渲染） |
| `/好感度排行` | 查看当前会话好感度排行榜（前10名，PIL 图片渲染） |
| `/好感度倒序` | 查看当前会话好感度倒序排行（低分在前，PIL 图片渲染） |
| `/重置好感度` | 重置自己的好感度 |

### 管理员指令
| 指令 | 说明 |
|------|------|
| `/设置好感度 <@用户> <分数>` | 强制设置指定用户好感度 |
| `/重置指定好感度 <@用户>` | 重置指定用户好感度 |
| `/禁言 <@用户> <秒数>` | 禁言指定用户（最大300秒） |
| `/解除禁言 <@用户>` | 解除指定用户禁言 |

## 禁言机制

当用户好感度≤-20 且出现持续恶劣行为（辱骂、骚扰、刷屏等）时，LLM 可在回复末尾输出 `[MUTE:N]` 标签触发禁言：

- **标签格式**：`[MUTE:秒数]`，秒数范围 1~300（最长 5 分钟）
- **禁言期间**：该用户的消息不会触发 LLM 回复，Bot 会随机发送"不理你了"、"假装听不见"等趣味回复
- **处罚力度**：
  - 轻度冒犯：20-60 秒
  - 中度冒犯：60-180 秒
  - 重度冒犯：180-300 秒
- **触发条件可自定义**：通过 WebUI 的 `mute_condition` 配置项自由定义禁言触发条件描述
- **管理员可手动干预**：使用 `/禁言` 和 `/解除禁言` 指令
- **可通过 WebUI 关闭**：`mute_enabled` 配置项

## 配置（WebUI）

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `favorability_enabled` | true | 好感度系统开关 |
| `sticker_enabled` | true | 表情包系统开关 |
| `mute_enabled` | true | 禁言系统开关 |
| `mute_condition` | "持续恶劣行为（如辱骂、骚扰、刷屏、恶意挑衅）" | 禁言触发条件描述，自由定义 |
| `system_time_enabled` | true | 系统时间注入（通过 extra_user_content_parts） |
| `user_info_enabled` | true | 用户信息注入（通过 extra_user_content_parts） |

## 表情包

在 `data/plugin_data/astrbot_plugin_favorability/stickers/<分类名>/` 下放置图片即可启用 `[STK:分类名]` 表情包功能。

支持格式：jpg、png、gif、webp

## 好感度等级

| 分数范围 | 等级 | 颜色 |
|----------|------|------|
| ≥ 70 | 挚爱 | 🔴 |
| 50 ~ 69 | 挚友 | 🟠 |
| 21 ~ 49 | 熟人 | 🟢 |
| -20 ~ 20 | 路人 | ⚫ |
| -50 ~ -21 | 生厌 | 🟡 |
| -70 ~ -51 | 憎恶 | 🔴 |
| < -70 | 仇敌 | ⚫ |
